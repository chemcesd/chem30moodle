SC 1.

 

Practice 1.

 

a. S in SO2

 

 

The oxidation number of S in SO2 is +4.

 

b. Cl in HClO4

 

 

The oxidation number of Cl in HClO4 is +7.

 

c. S in SO42–

 

 

The oxidation number of S in SO42– is +6.

 

d. Cr in Cr2O72–

 

 

The oxidation number of Cr in Cr2O72– is +6.

 

e. I in MgI2

 

The compound contains Mg2+ and I–.

 

I- is a monoatomic ion, so the oxidation number of I in MgI2 is –1.

 

f. H in CaH2

 

The compound contains Ca2+ and H– and is a hydride.

 

The oxidation number of H in CaH2 is –1.

 

Practice 2.   

 

a. N2O(g)                    2x + 1(–2) = 0            x = +1

 

b. NO(g)                     x + 1(–2) = 0              x = +2

 

c. NO2(g)                    x + 2(–2) = 0              x = +4  

           

d. NH3(g)                    x + 3(+1) = 0             x = –3

 

e. N2H4(g)                   2x + 4(+1) = 0           x = –2

 

f. NaNO2(s) contains Na+ and NO3-.

using NO3–,     x + 3(–2) = –1              x = +5

 

g. N2(g) is an element. The oxidation number is 0.

 

h. NH4Cl(s) contains NH4+ and Cl–.

using NH4+,    x + 4(+1) = +1            x = –3

 

Practice 3.   

 

a. C(s) (graphite) is an element, so the oxidation number is 0.

 

b. C6H12O6      6x + 12(+1) + 6(–2) = 0         x = 0

 

c. Na2CO3 contains Na+ and CO32–.

using CO32–,    x + 3(–2) = –2              x = +4

 

d. CO              x + 1(–2) = 0               x = +2

 

Practice 4. 

 

Remember that coefficients in an equation do not affect the value of oxidation numbers.

 

O in ClO3–, ClO2, and H2O is –2.

 

H in H+ and H2O is +1.

 

Cl in ClO3- is +5  (x + 3(–2) = –1).

 

Cl in Cl– is –1.

 

Cl in ClO2 is +4  (x + 2(–2) = 0).

 

Cl in Cl2 is 0 (element).

 

Practice 5.   

 

CH4                 x + 4(+1) = 0                       x = –4

 

CH3OH           x + 1(–2) + 4(+4) = 0              x = –2

 

CH2O              x + 2(+1) + 1(–2) = 0             x = 0

 

HCOOH          x + 2(+1) + 2(–2) = 0              x = +2

           

CO2                 x + 2(–2) = 0                      x = +4
